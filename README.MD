# Xbox SpeedTest (Node.js 实现)

## [解决 Xbox 在国内下载速度慢](https://baiyun.me/fix-slow-xbox-download-speed)

## 使用方法

安装

```shell
git clone https://github.com/isbasex/xbox-speed-test.git
```

运行测试

```shell
node index.js
```
